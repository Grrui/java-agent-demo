## 1.简介

JavaAgent是JDK1.5之后引入的新特性，此特性为用户提供了在JVM将字节码文件读入内存之后，JVM使用对应的字节流在Java堆中生成一个Class对象之前，用户可以对其字节码进行修改的能力， 从而JVM也将会使用用户修改过之后的字节码进行Class对象的创建。
更多的概念网上一大堆。
